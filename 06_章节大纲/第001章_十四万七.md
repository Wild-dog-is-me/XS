# 第001章 余额：章节大纲

## 正式正文位置
`第001章_重启.md`

## 本章作用
1. 用"余额"作为切入点，直接展示张有志已经赚到钱。
2. 通过自嘲式内心独白建立"废柴"人设。
3. 简短回忆交代穿越：2026年机房事故→2016年出租屋醒来。
4. 用加密货币和外包收入建立第一桶金的因果链。
5. 引入苏晚，通过消息互动建立感情线质感。
6. 确定日升科技方向，埋下面谈调研的尾钩。

## 场景列表
1. 出租屋：张有志看手机余额，147,832.56元。自嘲废柴。
2. 回忆（简短）：2026年机房事故，穿越到2016年3月。
3. 回到现在：加密货币赚钱过程，外包降维打击。
4. 苏晚消息：送饭、叫全名、猫咪表情包。
5. 笔记本计划：钱✓、方向、人、产品。
6. 写下"日升科技"。
7. 尾钩：明天去智能客服公司面谈，不是求职，是调研。

## 本章注意
1. 穿越回忆控制在300-500字，不要长篇大论。
2. 苏晚的出场通过消息，不要刻意制造戏剧冲突。
3. 自嘲和吐槽要有度，不能变成纯搞笑。
4. 结尾的孤独感要克制，不要煽情。
