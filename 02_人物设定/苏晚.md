# 苏晚：张有志的女友

## 基础信息
姓名：苏晚
年龄：22-23岁
职业：幼儿园教师
工作单位：某区公立幼儿园
与张有志关系：大学时期开始交往的女友，目前同居
首次出现：第001章（在场）

## 与张有志的感情状态（2016年）
1. 大学认识，恋爱一年多。感情基础好，但经济压力让两人关系有微妙的紧张。
2. 两人同居，住在张有志的出租屋里。苏晚月薪三千多，是主要经济来源。
3. 苏晚每天早上去幼儿园上班，晚上回来做饭。张有志在家写代码/接外包/投简历。
4. 同居生活有很多日常互动：苏晚催他吃饭、帮他收拾外卖盒、半夜起来给他盖被子、早上出门前在他桌上放一杯水。
5. 穿越后的张有志，对苏晚的态度有微妙转变：比穿越前更珍惜她，但也更不敢靠近她。

## 她不知道的事
1. 张有志穿越了。
2. 张有志账户里有十四万七。
3. 张有志借她的一万五被投入了加密货币。
4. 张有志上辈子和她分了手。

## 她在剧情中的作用
1. 情感锚点：张有志在穿越后的孤独和压力，需要一个真实的情感出口。
2. 现实拉力：苏晚代表"正常生活"的召唤。
3. 秘密压力：张有志不能对苏晚说穿越的事。
4. 细节载体：幼儿园的日常、两人之间的小互动，让小说有温度。
5. 成长对照：从"被动等待"到"主动参与"。

## 语言风格
说话轻，语速不快。喜欢用问句表达关心："吃了吗？""今天怎么样？"生气的时候不会大喊，而是用很平静的语气叫他的全名："张有志。"这种平静比任何争吵都让人心里发慌。

## 与张有志的互动模式
1. 日常关心：提醒吃饭、睡觉、出门带伞。
2. 安静陪伴：张有志在电脑前坐一整晚，苏晚就坐在旁边看书或备课。
3. 关键时刻的坚定：当所有人都质疑张有志时，苏晚可能是最后一个说"我相信你"的人。

## 禁止写法
1. 禁止写成工具人女友。
2. 禁止写成只会哭和等的被动角色。
3. 禁止写成"我什么都不懂但我支持你"的空洞人设。
4. 禁止让她成为主角的附属品。
