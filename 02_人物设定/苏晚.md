# 苏晚：张有志的女友

## 基础信息
姓名：苏晚
年龄：22-23岁
职业：幼儿园教师
工作单位：某区公立幼儿园
与张有志关系：大学时期开始交往的女友
首次出现：第001章（消息出场）

## 外貌气质
不是那种第一眼惊艳的类型。中等身高，偏瘦，习惯把头发扎成低马尾。眼睛不大，笑起来会弯成月牙。手指细长，指腹有常年和小朋友打交道留下的薄茧。

穿衣风格简单。平底鞋、牛仔裤、素色卫衣。不化妆，或者只涂一层很淡的润唇膏。身上偶尔会沾到小朋友画画时蹭上的颜料。

整体气质：安静、干净、温柔，但不是柔弱。

## 性格
1. 温柔但有主见。不会大吵大闹，但会在关键时刻说出自己的想法。
2. 细心，擅长观察情绪变化。张有志一个眼神的不对，她都能察觉。
3. 有耐心，职业训练出来的。面对三十个小朋友的幼儿园老师，情绪管理能力远超常人。
4. 偶尔有点小固执。她认定的事情不会轻易改变。
5. 不太会表达自己的需求，习惯先照顾别人。

## 欲望
1. 和张有志一起过上稳定的生活。
2. 希望张有志能找到一份"正经工作"，不用每天焦虑。
3. 深层：她希望张有志能对她真正敞开心扉。

## 恐惧
1. 怕张有志越来越陌生。穿越后的张有志和穿越前的张有志，在气质上会有微妙的不同。
2. 怕自己跟不上张有志的脚步。
3. 怕张有志做危险的事情。
4. 怕被排除在外。

## 与张有志的感情状态（2016年）
1. 大学认识，恋爱一年多。感情基础好，但经济压力让两人关系有微妙的紧张。
2. 苏晚已经在工作，月薪三千多。张有志还在找工作/刚毕业，经济上苏晚会接济他。
3. 两人见面频率不高，更多靠微信联系。苏晚发消息后面总喜欢跟猫咪表情包。
4. 穿越后的张有志，对苏晚的态度有微妙转变：比穿越前更珍惜她，但也更不敢靠近她。

## 她不知道的事
1. 张有志穿越了。
2. 张有志账户里有十四万七。
3. 张有志借她的一万五被投入了加密货币。
4. 张有志上辈子和她分了手。

## 她在剧情中的作用
1. 情感锚点：张有志在穿越后的孤独和压力，需要一个真实的情感出口。
2. 现实拉力：苏晚代表"正常生活"的召唤。
3. 秘密压力：张有志不能对苏晚说穿越的事。
4. 细节载体：幼儿园的日常、两人之间的小互动，让小说有温度。
5. 成长对照：从"被动等待"到"主动参与"。

## 语言风格
说话轻，语速不快。喜欢用问句表达关心："吃了吗？""今天怎么样？"生气的时候不会大喊，而是用很平静的语气叫他的全名："张有志。"这种平静比任何争吵都让人心里发慌。

## 与张有志的互动模式
1. 日常关心：提醒吃饭、睡觉、出门带伞。
2. 安静陪伴：张有志在电脑前坐一整晚，苏晚就坐在旁边看书或备课。
3. 关键时刻的坚定：当所有人都质疑张有志时，苏晚可能是最后一个说"我相信你"的人。

## 禁止写法
1. 禁止写成工具人女友。
2. 禁止写成只会哭和等的被动角色。
3. 禁止写成"我什么都不懂但我支持你"的空洞人设。
4. 禁止让她成为主角的附属品。
