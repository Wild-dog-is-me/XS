# 主角：张有志

## 基础信息
姓名：张有志
年龄：穿越后约23-25岁；心理年龄约32岁
教育背景：985顶尖高校计算机科学专业毕业，本科期间成绩优异，ACM竞赛拿过奖，导师评价"技术能力一流，就是人太闷"。毕业论文做的是分布式系统方向。
职业背景：未来头部AI公司的多模态/推理优化/平台工程核心研发
穿越时间：2016年5月21日
核心优势：未来认知、工程能力、技术路线判断、对AI产业周期的理解
核心短板：资金不足、人脉薄弱、商业谈判经验不足、容易用工程师思维低估组织和销售难度
感情状态：与苏晚交往一年多，感情稳定但受经济压力影响

## 外貌：普通到透明
黑发黑眸，身形消瘦，大众脸，存在感极低。丢在人群里绝对不会被多看一眼。

气质略带颓废，黑发乱糟糟的，刘海经常遮住眼睛。眼神大多数时候是空的，像一潭没有风的水。但偶尔——极偶尔——会流露出一种与年龄不符的孤独感。那种孤独不是装出来的，是真的见过太多事情之后沉淀下来的。

长相其实清秀，但总被忽视。大学同学叫他"背景板"，同事叫他"那个谁"，房东叫他"302的"。他习惯了。

## 表面：废柴、怂萌、爱吐槽
### 自卑懦弱
遇事退缩，怕麻烦，习惯自嘲"废柴""衰小孩""衰仔"。他的自嘲不是表演，是真的觉得自己不行。面试的时候会紧张到手抖，和陌生人说话会不自觉地低头，被人夸奖会浑身不自在。

### 嘴炮吐槽
紧张的时候烂话不断，自称"自动吐槽机"。用玩笑掩饰尴尬与脆弱。他的吐槽不是毒舌，更像是一种自我保护机制——只要我先嘲笑自己，别人就伤害不了我。

经典语录风格：
- "我就是个废柴，废柴能有什么坏心思呢？"
- "我不是不想努力，我是怕努力了还是这个鸟样。"
- "这个世界的bug大概就是我吧。"

### 消极佛系
不爱出风头，怕惹事，日常躺平，缺乏自信。他的简历上写着"抗压能力强"，实际上他的抗压方式是假装压力不存在。别人卷的时候他在发呆，别人焦虑的时候他在吃泡面。不是因为他不在乎，是因为他在乎的事情太多了，多到不知道该先在乎哪一个。

## 内里：孤独敏感、重情重义
### 深度孤独
从小寄人篱下，父母在外地打工，他跟着爷爷奶奶长大。缺爱缺关注，内心极度敏感，缺安全感。他不是不想交朋友，是不知道怎么交朋友。他有一种与生来的疏离感，像是被一层看不见的玻璃罩着，能看见外面的世界，但触碰不到。

这种孤独在他穿越之后变得更加深刻。他知道了未来会发生什么，但不能告诉任何人。这种"我知道一件所有人都不知道的事"的感觉，比任何孤独都更孤独。

### 极度珍视善意
对在乎的人愿意付出生命守护，讲义气，有担当。苏晚给他买一杯奶茶，他能记一个月。同事帮他带一次外卖，他能帮人家修三天bug。他对善意的敏感程度像一台校准过度的传感器——任何一点温暖都会被他放大一百倍。

### 善良心软
共情力强，见不得别人受苦，对伤害自己的人也难下狠手。他会在路边给流浪猫买火腿肠，会在同事被骂的时候偷偷帮忙改代码，会在苏晚加班到很晚的时候煮一碗面放在桌上然后假装什么都没做。

## 成长弧光：从逃避到承担
张有志是一个矛盾的结合体。

他是那个遇事就退缩的废柴，也是那个在绝境中咬牙往前走的人。他是那个连面试都紧张的怂货，也是那个把全部积蓄押进加密货币然后一声不吭等结果的赌徒。

他的成长不是突然变强，而是一次又一次地在"我不行"和"但我必须做"之间挣扎，然后每一次都选择了后者。

他是每个普通人心中都有的衰小孩，也是绝境中觉醒的英雄。

## 穿越后的变化
1. 比穿越前更沉默。脑子里同时装着2016年和2026年两个世界的信息，像一台内存不足的机器。
2. 比穿越前更珍惜苏晚。上辈子的记忆让他知道她有多重要，但也让他更不敢靠近——因为他不确定自己能不能给她一个稳定的未来。
3. 眼神偶尔会失焦。那是他在回忆2026年的事情。苏晚会注意到这个变化，但说不上来哪里不对。
4. 嘴上的自嘲没变，但内心比以前更坚定。废柴的壳还在，但壳里面多了一个方向。

## 与苏晚的关系
1. 大学认识，恋爱一年多。感情基础好，但经济压力让两人关系有微妙的紧张。
2. 苏晚已经在工作，月薪三千多。张有志还在找工作/刚毕业，经济上苏晚会接济他，这让他的自尊心很不舒服。
3. 他不会对苏晚说"我爱你"，但会在她睡着后帮她盖好被子，会在她加班的时候煮面，会在深夜给她发一条消息："明天降温，多穿点。"
4. 穿越后他最大的秘密不是华天科技，不是加密货币，而是"我不是原来的张有志"。这个秘密像一根刺，扎在他心里最柔软的地方。

## 语言风格
1. 日常：自嘲、吐槽、碎碎念。"我就是个废柴""这个世界的bug就是我""妈的，连植物都活得比我有精神"。
2. 认真时：简短、直接、不废话。技术讨论时会突然变得很专注，像换了一个人。
3. 对苏晚：语气会软一点。不多，但苏晚能听出来。
4. 内心独白：丰富、细腻、偶尔文艺。表面是个废柴，内心是个诗人——虽然他自己不知道。

## 能力边界
1. 不能凭空记住所有细节。大方向记得，具体数字模糊。
2. 不能一个人完成所有技术。他是个废柴，废柴需要队友。
3. 不能让所有人立刻相信他。他连自己都不太相信。
4. 需要通过原型、数据、结果赢得信任。嘴上说的不算。
5. 不能对苏晚说穿越的真相。这个秘密会一直压着他。

## 成长线
废柴 → 勉强能用的人 → 能扛事的人 → 能带队伍的人 → 真正的创业者

感情线成长：独自扛 → 学会依赖苏晚 → 学会部分坦诚 → 学会在事业和感情之间找平衡
