# 商业化路径

## 早期现金流
1. 企业模型部署工具授权。
2. AI项目工程化咨询，但不能陷入纯外包。
3. 开发者工具订阅。
4. 私有化部署收费。

## 中期增长
1. 团队版代码助手。
2. 企业知识库与RAG平台。
3. 模型评测和推理优化服务。
4. 与云厂商/IDC合作。

## 后期平台化
1. AgentOS平台费。
2. 模型调用按量计费。
3. 企业插件生态分成。
4. 行业解决方案。

## 商业冲突
- 客户要求私有化，但公司希望SaaS规模化。
- 大客户定制需求拖慢产品路线。
- 投资人要求增长，技术团队要求重构。
- 免费开源竞品带来价格压力。

## 写作原则
每一笔订单都要有销售过程、客户顾虑、试用结果和交付代价。不要让客户只因主角一句话就签合同。
