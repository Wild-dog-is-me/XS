# 第001章 余额

张有志盯着手机屏幕上那个数字看了三分钟。

**147,832.56元。**

他想，如果三个月前有人告诉他，他会坐在出租屋的床边，用一台屏幕裂了纹的手机，看着自己账户里有十四万——他一定会觉得那个人在讲笑话。

三个月前，这个数字是4,871.03元。

准确地说，银行卡里4,871.03元，信用卡欠八千，房租还欠着一个月，手机里还有三条催缴话费的短信。

那个时候他还不是这个张有志——不，他一直是张有志，只是他记得的那个张有志，是2026年的张有志。三十二岁，在一家AI公司做推理优化和平台工程，月薪三万五，租着一间一居室，冰箱里永远只有可乐和速冻水饺，偶尔想起来吃饭已经是凌晨两点。

然后有一天凌晨，他在公司机房外面处理线上故障。

prod-mm-infer-east-02，多模态推理服务，A区第三排机柜温度越过了阈值。他把请求切到备用集群，改了调度策略，把长上下文任务拆成两段。终端里的日志在滚：

`route updated`

`worker drained`

`cache rebuilt`

然后机房方向传来一声闷响。

走廊灯闪了一下。

屏幕上的曲线停住了。

下一秒，所有声音都被抽走。

——再睁开眼的时候，他躺在一间陌生的出租屋里。天花板上是一盏发黄的旧吸顶灯，墙角有受潮的暗斑，窗外传来电动车经过减速带的声音。

手机屏幕上的日期是2016年3月15日。

他花了一整天确认自己不是在做梦，不是在精神病院，而是真的回到了十年前。

2016年。

距离AlphaGo击败李世石刚过去两周。

距离Transformer论文发表还有一年半。

距离BERT发布还有两年半。

距离ChatGPT引爆全球还有六年半。

他有十年的时间。

但当时张有志的第一反应不是"我要改变世界"。

他的第一反应是：

**卧槽，我房租怎么办？**

这就是张有志。

一个废柴穿越者的真实开局。没有系统，没有金手指，没有开局一百亿。只有四千八百七十一块零三分，一张欠了八千的信用卡，和一个正在等他回消息的女朋友。

——对，女朋友。

苏晚。

想到这个名字，张有志的手指在屏幕上停了一下。

算了，先不想这个。先说钱。

他怎么从四千八百块钱变成十四万的？

答案很简单：炒币。

2016年的比特币，价格大约在四百美元左右。以太坊更便宜，十几美元一个。他记得2017年底比特币会涨到接近两万美元，记得以太坊会在2017年涨到一千多美元。

所以他做了一件他上辈子想都没想过的事：他把所有的钱——四千八百七十一块零三分——全部买了比特币。然后用信用卡套现了八千。然后找苏晚借了一万五。

苏晚问他借钱干什么。

他说交房租。

苏晚没有追问。她只是把钱转过来，然后发了一条消息："不用急着还，好好吃饭。"

张有志看着那条消息，想了很久。

最后他打了三个字："谢谢你。"

苏晚回了一个猫咪的表情包。

他把那三万多块全部扔进了加密货币。

然后就是等。

等的过程中，他每天都在骂自己：

"你是不是傻？万一你记错了呢？万一时间线不一样呢？万一这是平行宇宙呢？"

"你一个废柴程序员，连bug都修不利索，你炒什么币？"

"你就不怕把苏晚的钱也赔进去？到时候你怎么跟她交代？'对不起晚晚，我把你的钱赔光了，因为我以为自己穿越了？'你觉得她会信吗？"

但他还是没卖。

不是因为他有信心。是因为他怂。他太怕卖了之后涨上去会后悔，所以干脆就不动。用他自己的话说："我不是坚定持有者，我只是不敢操作。"

结果，比特币从四百涨到了六百多。

以太坊从十几块涨到了七八十。

他又陆续做了一些外包——帮人写爬虫、做数据分析、接AI相关的私活。2026年的技术能力放在2016年，简直是降维打击。一个简单的图像分类项目，他用2016年的框架就能做出2020年的效果。

他的账户余额从四千八变成了十四万七。

张有志把手机放下，靠在床头。

出租屋的灯泡是那种最便宜的LED灯泡，光线发白，照得人脸像纸一样。房间很小，十几平米，靠墙一张折叠桌，桌上放着一台黑色笔记本电脑，旁边堆着几本书和一盆已经枯死的绿萝。

那盆绿萝是他搬进来的时候就在的。他忘了浇水，然后它就死了。

张有志看着那盆绿萝，忽然想笑。

"妈的，"他自言自语，"连植物都活得比我有精神。"

手机震了一下。

他拿起来看。

是苏晚的消息。

"今天怎么样？有没有去面试？"

后面跟了一个猫咪打哈欠的表情包。

张有志看着那个表情包，嘴角的弧度变了一点。

苏晚。他的女朋友。一个幼儿园老师。

她不知道他穿越了。她不知道他账户里有十四万七。她不知道他正在计划做一件可能改变整个行业的事情。

她只知道他最近很忙，很少回消息，经常不吃饭。

"还没。"他回了两个字。

"又没吃早饭？"

"吃了。"

"吃的什么？"

"泡面。"

"张有志。"

苏晚发了他的全名。他知道这意味着什么——她生气了。苏晚生气的时候不会大喊大叫，她只会用很平静的语气叫你的全名。这种平静比任何争吵都让人心里发慌。

"我晚上给你送饭。"苏晚说。

"不用——"

"六点半。你在家等着。"

张有志看着屏幕，手指悬在键盘上方，想打点什么。

他想说"对不起"。

对不起什么？对不起我穿越了？对不起我不是原来的张有志？对不起我上辈子跟你分了手，这辈子想重新来过，但我连自己都养不活？

他只打了一个字："好。"

苏晚回了一个猫咪点头的表情包。

张有志把手机放下，从枕头底下摸出一个旧笔记本。

封面是黑色的，边角已经卷了。他翻开第一页，上面写着几行字，字迹歪歪扭扭的：

**2016.3.18，重启。**

**目标：先活下来。然后，做一家能活到大模型时代的公司。**

他盯着这几行字看了一会儿。

三个月前写这些的时候，他账户里只有四千八百块钱。现在他有十四万七。

依然不多。但至少——他证明了一件事：他的记忆是准确的。未来会发生的事情，确实在发生。

AlphaGo击败李世石。这个事件在他的记忆里存在，在现实中也发生了。

这意味着Transformer会来。BERT会来。GPT会来。ChatGPT会来。

一切都会来。

他只需要活到那一天。

张有志翻到下一页，开始写计划。

**第一步：钱。** ✓ 已完成。147,832.56元。不多，但够启动。

**第二步：方向。** AI工程化工具。不造大模型——没钱，没人，没卡。先做基础设施：模型部署、推理服务、代码智能。2016年的AI行业还在做知识图谱和规则引擎，没有人知道Transformer是什么。这是优势。

**第三步：人。** 需要合伙人。一个懂系统的，一个懂产品的，一个能跑商务的。但问题是——他是张有志。一个存在感为零的废柴。废柴怎么说服别人跟他干？

答案是：不知道。先做再说。

**第四步：产品。** 先做一个最小可用的东西。代码搜索？模型部署工具？数据管线？

写到这里，他停住了。

笔尖在纸上留了一个小小的墨点。

他在墨点旁边写下四个字：

**华天科技。**

不玄，也不酷。像一家会出现在写字楼名录、采购合同和企业付款审批单上的公司。

这很好。

公司不是写给朋友圈看的，是要活在合同、发票、代码仓库和客户故障群里的。

张有志合上笔记本，站起来。

窗外的天已经开始暗了。城中村的巷子里，路灯一盏一盏亮起来，照着一面斑驳的墙。远处有狗叫声，近处有水管滴水的声音。

他走到窗边，深吸一口气。

2016年的空气。

比2026年的干净一点。

也比2026年的孤独一点。

因为他知道未来会发生什么，但不能告诉任何人。

这种孤独不是没有人陪的孤独。是有人陪——有一个会给他送饭、会叫他全名、会发猫咪表情包的女朋友——但他不能对她说真话的孤独。

他不能说"我是从2026年穿越回来的"。

他不能说"我知道未来十年会发生什么"。

他甚至不能说"我账户里有十四万七"。

因为在苏晚眼里，他还是那个到处投简历、找工作、偶尔接点外包的废柴程序员。

张有志转过身，重新拿起笔记本。

在"华天科技"四个字下面，他写了一行小字：

**明天上午十点，去看看那家AI公司。**

明天有一场面试。智能客服方向的算法工程师岗位。

他不打算去上班。

他打算去看看——2016年的AI公司到底长什么样。他们卡在哪里，缺什么，怕什么，愿意为什么付钱。

这些信息，比十四万七千块钱更有价值。

张有志合上笔记本，关了灯。

躺在床上，盯着天花板。

黑暗中，他忽然想起一件事。

上辈子，他和苏晚在2019年分了手。

原因很简单：他太忙了，她太累了，两个人的生活轨迹慢慢错开，最后谁都没有力气再拉对方一把。

分手那天，苏晚没有哭。她只是很安静地说了一句："有志，你以后记得吃饭。"

然后她就走了。

张有志在出租屋的沙发上坐了一整夜，没有哭，也没有追。

这就是他。一个连分手都不敢挽留的废柴。

但这一次——

他不确定自己能不能做得更好。

他甚至不确定自己有没有资格重新来过。

因为他知道未来会发生什么，却不知道自己配不配得上这个未来。

手机屏幕亮了一下。

苏晚发来一条消息：

"明天降温，记得多穿点。晚安。"

张有志看着那条消息，忽然觉得眼睛有点酸。

他回了两个字：

"晚安。"

然后他把手机翻过去，屏幕朝下，放在枕头旁边。

黑暗中，他的眼睛亮了一下。

不是兴奋。

是一个废柴下定决心时的一点点光。

很微弱。

但确实存在。
