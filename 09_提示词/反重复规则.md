# 反重复规则

## 问题背景
第016-024章出现了严重的重复问题：结尾模板化、人物反应单一、剧情线循环。这些规则是为了防止再次出现这种情况。

## 一、禁止使用的重复句式（黑名单）

以下句式已经在前面的章节中使用过多次，禁止再使用：

### 结尾模板（禁止）
- "张有志看着窗外的科技园。夜色里，那些办公室的灯光一盏一盏地灭了。只有几家公司的灯还亮着，包括深度求索。"
- "这个市场正在以前所未有的速度膨胀。而他，已经占住了入口。"
- "但他知道，真正的战斗，才刚刚开始。"
- "张有志看着窗外的科技园。夜色越来越深，但他知道，这个市场正在以他上辈子见过的最快速度膨胀。"
- 任何形式的"看着窗外→感慨市场→真正的战斗才刚开始"三段式结尾
- "特斯拉vs二手本田"对比场景（已经用过3次，禁止再用）
- "他走向自己的二手本田，打开车门，坐进去，发动引擎"（禁止再用）

### 人物反应模板（禁止）
- "韩洲点点头。"（同一章内最多出现2次）
- "韩洲看着他。"（同一章内最多出现1次）
- "韩洲的眼睛亮了。"（同一章内最多出现1次）
- "韩洲长出了一口气。"
- "操，吓死我了。"
- "张有志看着他。"（同一章内最多出现2次）
- "张有志的手指在桌上敲了一下。"

### 对话模板（禁止）
- "你这么确定？" → "确定。"
- "那我们走着瞧。"
- "那倒是。"
- "好。那我们继续跑。"
- "比如？" → 列举几个方向

### 感情线模板（禁止）
- 每章结尾都给苏晚发消息
- "苏晚秒回"
- "红烧排骨"/"红烧肉"（同一章内最多出现1次）
- "苏晚总是知道什么时候该做什么菜"
- 苏晚永远在厨房做饭
- 苏晚永远是"我信你""我等你""傻瓜"
- 苏晚永远没有自己的情绪波动
- 每次都是张有志主动发消息，苏晚被动回复
- 苏晚对张有志的变化只是"我信你"——太简单了
- 苏晚永远是支持者，没有质疑、没有担心、没有愤怒
- **感情线情节不能重复**：不能每章都是"苏晚想学UI设计+学费八千+拒绝张有志出学费"，要有新的冲突和发展

### 剧情线模板（禁止）
- "顾承安不死心"（同一章内禁止出现）
- 顾承安每章都来搞事
- 每章都是"客户签约→顾承安搞事→张有志碾压"的循环

## 二、结尾规则

### 每章结尾必须：
1. 是独特的，不能用模板
2. 留下具体的悬念（不是泛泛的"战斗才刚开始"）
3. 与下一章的内容直接相关
4. 用具体物件、具体动作、具体对话收尾，不要用感慨收尾

### 结尾禁止：
1. 禁止用"看着窗外"收尾
2. 禁止用"市场膨胀"收尾
3. 禁止用"真正的战斗才刚开始"收尾
4. 禁止用苏晚的消息收尾（偶尔可以用，但不能每章都用）
5. 禁止用中二热血漫画式收尾（见下方详细说明）

### 结尾反中二规则（强制执行）

**问题诊断：**
我写的结尾太中二——像热血漫画的旁白，不像现实主义小说。

**禁止的中二结尾：**
1. 禁止"而他们，只有钱"这种排比对比
2. 禁止"他不怕""他不会输""他一定会赢"这种热血宣言
3. 禁止"因为他有XXX"这种"金句"式收尾
4. 禁止任何升华、煽情、鸡汤式结尾
5. 禁止用感慨收尾（"他忽然明白了""他终于懂了"）

**好的结尾应该是：**
1. 具体的、有画面感的
2. 不煽情、不升华、不中二
3. 留白，让读者自己去想
4. 用动作、物件、对话收尾，不用感慨
5. 平淡、克制、真实

**好的结尾示例：**
- 他关掉电脑，站起来，走到窗边。楼下有人在搬服务器。
- 手机响了。他看了一眼屏幕，是一个陌生的号码。
- 韩洲发来一条消息："出事了。"
- 他回到家，苏晚已经睡了。桌上留着一碗汤，还热着。
- 他躺在床上，看着天花板。明天还要早起。

**坏的结尾示例（禁止）：**
- 而他们，只有钱。
- 他不怕。因为他有技术，有团队，有客户。
- 创业和爱情，不能兼得？他不信。
- 这个市场正在以前所未有的速度膨胀。而他，已经占住了入口。

### 结尾示例（好的结尾）：
- 一个电话打进来，屏幕上显示一个陌生的名字
- 张有志看着桌上的合同，忽然发现一个数字不对
- 韩洲发来一条消息："出事了。"
- 办公室的门被推开，一个他们没想到的人走了进来

## 三、人物反应多样化规则

### 韩洲的反应不能总是：
- 点头
- 看着张有志
- 眼睛亮了
- 长出了一口气

### 韩洲可以有以下反应：
- 沉默不语，继续敲代码
- 皱眉，提出质疑
- 笑了，但笑得很勉强
- 摇头，表示不同意
- 拍桌子，表示愤怒
- 站起来走来走去，表示焦虑
- 低头看手机，表示心不在焉
- 问一个尖锐的问题
- 岔开话题
- 沉默很久才说话

### 其他人物同理：
- 许知夏不能总是"翻开活页纸写数字"
- 老刘不能总是"沉默几秒然后点头"
- 孙鹏不能总是"说话很快"

## 四、剧情线推进规则

### 顾承安的剧情线不能：
- 每章都来搞事
- 每次都被拒绝
- 每次都说"走着瞧"
- 每次都失败

### 顾承安的剧情线应该：
- 有真正的进展（不是原地踏步）
- 有成功的时刻（不能每次都失败）
- 有变化（不是同一个套路重复）
- 与其他剧情线交织（不能独立存在）
- **顾承安是腾讯系的人，他有资源、有人脉、有钱，他应该是一个真正的威胁，不是一个笑话**

### 顾承安必须有的成功：
1. **至少抢走一个客户**：不能每次都被张有志守住
2. **至少挖走一个人**：不能每次都被张有志留住
3. **至少让张有志损失一次**：不能每次都无成本获胜

### 客户签约不能：
- 每章都签一个新客户
- 每次都是"张有志用技术碾压→客户被征服→签约"
- 每次都是同样的流程
- **每章都赚钱**

### 客户签约应该：
- 有不同的挑战（不是每次都靠技术碾压）
- 有失败的可能（不能每次都成功）
- 有意外（不能按计划进行）
- 有代价（不能无成本获得）
- **过程可以拉长**：一个客户的谈判可以跨2-3章
- **有些客户会流失**：不能只进不出

### 剧情波折规则（强制执行）

**问题诊断：**
剧情太顺利——每章都签合同赚钱，对手太弱，没有真正的失败和损失。

**强制要求：**
1. **每5章必须有至少1章是失败/损失/僵持**：不能每章都有好结果
2. **每10章必须有至少1个客户流失**：不能只进不出
3. **每10章必须有至少1次顾承安成功**：不能每次都输
4. **每10章必须有至少1次团队危机**：不能一直稳定

**允许的剧情类型：**
1. **失败**：客户被抢走、项目失败、资金紧张
2. **僵持**：谈判没有结果、等待回复、悬而未决
3. **损失**：团队成员离开、客户流失、技术故障
4. **代价**：为了一个客户放弃另一个、为了赶工牺牲质量
5. **等待**：有些章节是等待、准备、调整，不是每章都有结果

**禁止的剧情：**
1. 禁止每章都签合同赚钱
2. 禁止每章都有好结果
3. 禁止顾承安每次都失败
4. 禁止没有真正的损失和代价
5. 禁止一切按计划进行

## 4.5 感情线规则（重要！）

### 问题诊断
苏晚目前的角色是"送饭NPC"——永远在厨房做饭，永远说"我信你"，永远没有自己的生活。这不是一个真实的人物，这是一个工具人。

### 苏晚必须有自己的生活
1. **工作**：苏晚有自己的工作（设定是某公司职员），不能永远在家做饭
2. **朋友**：苏晚有自己的社交圈，不能只有张有志
3. **烦恼**：苏晚有自己的烦恼（工作压力、家庭问题、对未来的迷茫）
4. **兴趣**：苏晚有自己的爱好（不是只有做饭）
5. **情绪**：苏晚会生气、会焦虑、会失望、会开心，不是永远温柔

### 感情线必须有冲突
1. **信任危机**：苏晚发现张有志的变化，不能只是"我信你"，应该有怀疑、追问、争吵
2. **时间冲突**：张有志创业太忙，陪苏晚的时间变少，苏晚会不满
3. **价值观冲突**：苏晚可能不理解张有志为什么要冒险创业
4. **秘密压力**：张有志不能说实话（穿越），这个秘密会成为两人之间的隔阂
5. **第三方介入**：不是狗血三角恋，而是苏晚的家人/朋友对张有志的质疑

### 感情线必须有进展
1. **不能每章都是"发消息→秒回→红烧排骨"**
2. **感情线要有起伏**：热恋→矛盾→和好→更深的理解
3. **苏晚要参与剧情**：不能只是旁观者，她可以帮忙、可以阻碍、可以提供关键信息
4. **苏晚要有成长**：从不理解到理解，从担心到支持，从被动到主动

### 感情线场景多样化
1. **不能每次都是出租屋做饭**
2. 可以在：公园散步、咖啡馆聊天、公司楼下等、医院（生病）、车站（离别）、电话吵架、微信冷战、朋友聚会、家庭聚餐
3. **苏晚可以主动**：给张有志送饭、去公司找他、帮他处理杂事、提醒他休息
4. **苏晚可以被动**：等他回家、担心他的安全、不理解他的选择

### 感情线禁止项
1. 禁止苏晚永远是"我信你"
2. 禁止苏晚永远在厨房做饭
3. 禁止每章结尾都给苏晚发消息
4. 禁止苏晚对张有志的变化没有任何质疑
5. 禁止苏晚没有自己的生活
6. 禁止感情线没有任何冲突
7. 禁止苏晚只是"送饭NPC"

### 感情线必须有的元素
1. **真实的对话**：有打断、有跑题、有语气词、有答非所问
2. **真实的情绪**：有生气、有焦虑、有失望、有开心、有委屈
3. **真实的冲突**：有质疑、有争吵、有冷战、有和好
4. **真实的细节**：有具体的物件、具体的动作、具体的环境
5. **真实的关系**：有依赖、有独立、有妥协、有坚持

### 对话写法规则（强制执行）

**问题诊断：**
我写的对话太僵硬——"苏晚说""苏晚问""苏晚看着他"，像在写报告，不像在写情侣。

**禁止的对话写法：**
1. 禁止每句话都用"苏晚说""苏晚问"开头
2. 禁止每句话都完整、礼貌、有逻辑
3. 禁止对话像两个陌生人在谈话
4. 禁止没有昵称、没有语气词、没有打断
5. 禁止没有肢体语言、没有自然反应

**必须有的对话元素：**
1. **昵称**：宝贝、老公、亲爱的、笨蛋、傻瓜（根据场景选择）
2. **语气词**：嗯、啊、那个、呃、哎、嘛、啦、呀
3. **打断**：话没说完就被打断
4. **省略**：说一半不说了
5. **肢体语言**：靠在肩膀上、拉手、捏脸、戳胳膊、踢脚
6. **答非所问**：问A答B
7. **跑题**：说着说着就跑偏了
8. **自然反应**：笑、叹气、沉默、翻白眼、撇嘴

**好的情侣对话示例：**
```
"你今天怎么这么晚？"
"嗯……公司有点事。"
"又是公司。"苏晚撇嘴，"你公司比我还重要。"
"没有。"张有志伸手想搂她，她躲开了。
"别碰我，我生气了。"
"宝贝……"
"谁是你宝贝。"苏晚站起来，走到厨房，"吃饭了，菜凉了。"
```

**坏的情侣对话示例（禁止）：**
```
"你今天怎么这么晚？"苏晚问。
"公司有点事。"张有志说。
"你公司比我还重要。"苏晚说。
"没有。"张有志说。
"那你为什么每天这么晚？"苏晚问。
"因为创业很忙。"张有志说。
```

## 五、写前检查清单（新增）

在写每一章之前，必须：
1. 读取上一章的最后100行，确认当前状态
2. 检查本章是否使用了黑名单中的句式
3. 确认本章的结尾是独特的
4. 确认本章的人物反应是多样化的
5. 确认本章的剧情线有真正的推进（不是原地踏步）

## 六、章节结构多样化规则（强制执行）

### 核心原则
**每5章中必须有至少1章是以情感线/人物内心为主线的章节，不能全是"签合同"。**

### 强制轮换规则
以下类型的章节必须轮换出现，不能连续3章都是同一类型：

| 章节类型 | 说明 | 频率要求 |
|---|---|---|
| 商业线 | 签合同、谈客户、竞争、融资 | 最多连续2章 |
| 情感线 | 苏晚、感情冲突、家庭、内心 | 每5章至少1章 |
| 团队线 | 招人、磨合、冲突、成长 | 每5章至少1章 |
| 技术线 | 产品迭代、bug、技术突破 | 每5章至少1章 |
| 危机线 | 失败、危机、转折、代价 | 每5章至少1章 |

### 强制执行：每5章必须包含
1. **至少1章以情感线为主线** — 苏晚的戏份必须超过50%
2. **至少1章主角没有赢** — 不能每章都碾压对手
3. **至少1章有意外/转折** — 不能一切按计划进行

### 章节类型检查清单
写每一章之前，必须：
1. 确认本章属于哪种类型（商业/情感/团队/技术/危机）
2. 检查最近3章是否都是同一类型
3. 如果最近2章都是商业线，本章必须换类型
4. 如果最近5章没有情感线章节，本章必须是情感线

### 章节结构模板（禁止使用）
❌ 开头：某个事件发生 → 中间：张有志用技术碾压 → 结尾：韩洲问下一步，张有志说继续跑

### 章节结构模板（可以使用）
✅ 倒叙：从结果开始，回溯过程
✅ 多线并行：两条剧情线同时推进
✅ 悬念开头：从一个谜团开始
✅ 冲突开头：从一个危机开始
✅ 情感线为主：以苏晚/感情线为主线
✅ 内心独白为主：以张有志的内心世界为主线
✅ 失败的章节：主角没有赢
✅ 转折的章节：剧情方向发生重大变化

### 情感线章节的具体要求
当本章是情感线章节时，必须满足：
1. 苏晚的戏份超过50%
2. 有至少一场完整的对话（不是几句话就结束）
3. 有至少一个情绪转折点
4. 展现苏晚自己的生活/工作/烦恼
5. 推动两人关系的进展（不是原地踏步）
6. 不能以"签合同"结尾

## 七、字数节奏规则（补充）

### 每章3000-4500字，但不能：
- 用重复的内容凑字数
- 用模板化的结尾收尾
- 用"看着窗外"拖延篇幅

### 每一段必须：
- 推进剧情
- 或传递新信息
- 或展现人物性格
- 如果删掉不影响上下文，必须删掉
