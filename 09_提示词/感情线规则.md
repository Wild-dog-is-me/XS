# 感情线规则

## 感情线定位
感情线和创业线是双主线，不是主次关系。读者看创业爽，看感情也爽，两条线互相推动。创业的压力会影响感情，感情的温度会影响创业决策。

## 同居设定
苏晚和张有志同居。这意味着：
1. 苏晚不是"偶尔出场送饭"的角色，她是每天都在场的人。
2. 张有志写代码到凌晨两点，苏晚就在旁边睡着了，被子滑到地上，他起来给她盖上。
3. 苏晚早上去上班前，会在桌上放一杯水，有时候附一张便签："记得吃饭。"
4. 张有志赚了钱但不能告诉苏晚来源，这种秘密在同居生活中会被放大——苏晚会发现他最近买东西不心疼了，但又说不清为什么。
5. 同居意味着争吵也是日常。苏晚嫌他不收拾房间，张有志嫌她把他的技术书挪了位置。但吵完之后，苏晚还是会给他留一盏灯。

## 苏晚的角色
苏晚不是花瓶，不是工具人。她是一个在自己领域里认真工作的独立个体。她的价值不来自张有志的成功，来自她自己。

## 感情线写法
1. 用细节写感情，不用宣言。不说"他很爱她"，写"他把面端到桌上，筷子摆好，然后回去继续看电脑"。
2. 用沉默写张力。苏晚不追问，张有志不解释。两个人坐在同一个房间里，各自沉默。
3. 用日常写真实。一起吃路边摊、一起挤公交、一起在超市里比较哪袋米更便宜。
4. 用秘密写悬念。张有志不能对苏晚说穿越的事。这个秘密会像一根刺。
5. 用成长写结局。苏晚不是一直被动等待。

## 感情线节奏
- 前期（第一卷）：同居日常。苏晚是张有志生活中最稳定的存在。张有志的秘密（穿越、赚钱）在同居生活中开始产生裂缝。
- 中期（第二卷）：苏晚发现张有志的变化越来越大，开始不安和追问。
- 高潮（第三卷/第四卷）：信任危机爆发。
- 后期：两人在危机中重新建立信任。

## 与新主角人设的适配
张有志是"废柴"型主角，他对苏晚的感情表达方式是：
1. 嘴上自嘲"我就是个废柴"，但会默默记住苏晚说过的每一句话。
2. 不会说"我爱你"，但会在深夜把滑到地上的被子捡起来给她盖上。
3. 苏晚叫他全名"张有志"的时候，他知道她生气了，会怂，但不会逃避。
4. 他对苏晚最大的温柔是：不让她知道自己穿越的事，因为他不想让她担心。

## 禁止写法
1. 禁止写成"男主创业成功，女主在旁边鼓掌"。
2. 禁止写成"女主只会哭和等"。
3. 禁止用"我什么都不懂但我支持你"来代替真实的情感互动。
4. 禁止让苏晚变成只在章节结尾出场的"送饭NPC"。
5. 禁止让感情线变成言情小说。
